🌐 ENRZ-DP50 디스플레이 장치

핵심 기능 중심 퀵 매뉴얼
ENRZ-DP50은 Z50 시리즈 서보 너트러너 및 서보 프레스 시스템의 운영을 혁신하는 강력한 휴먼-머신 인터페이스(Human-Machine Interface, HMI)입니다. PC 없이 현장에서 실시간 모니터링, 설정 변경, 데이터 분석 및 직접 제어를 가능하게 하여 생산성과 관리 효율성을 극대화합니다.

🚀 1. 핵심 기능 활용

ENRZ-DP50은 현장 중심의 운영 및 관리를 위한 강력한 기능들을 제공합니다.

A. 실시간 모니터링 기능

PC 없이 생산 현황과 장비 상태를 즉시 파악합니다.

📊 I/O 모니터

입출력 신호 ON/OFF 상태를 실시간 확인하며, 출력을 강제 출력하는 기능을 통해 초기 설정 및 문제 해결 시간을 획기적으로 단축합니다.

📈 체결/가압 결과 모니터

작업 완료 후의 최종 결과 데이터를 상세 조회 및 분석합니다.

🔍 NG 이력 모니터

불량(NG) 이력 및 시스템 에러 이력(최대 128건)을 상세히 저장하여 신속한 원인 분석 및 조치를 지원합니다.

B. 파라미터 편집 (현장 설정 변경)

현장에서 즉시 작업 조건을 변경하여 생산 유연성을 높입니다.

기본 절차: READ (읽기) → 항목 선택/값 변경 → WRITE (쓰기)의 3단계 프로세스
  • 체결 프로그램 편집: 너트러너의 목표 토크, 각도, 속도 등 핵심 작업 조건 정밀 편집
  • SPU50 프레스 파라미터 편집: 프레스의 목표 압력(하중), 정지 위치, 속도 설정

C. 데이터 시각화 및 분석

품질 관리를 위해 정량적 데이터뿐만 아니라 시각적 그래프 분석을 제공합니다.

📉 토크/프레스 커브 표시

  • 너트러너: '토크-각도' 또는 '토크-시간' 커브
  • 프레스: '하중-위치' 또는 '하중-시간' 커브
  • 최대 4개 축 데이터 동시 표시로 작업 편차 비교 분석 가능

D. 직접 운전 제어

자동 운전 중의 문제 발생이나 테스트 시 화면 터치를 통해 시스템을 직접 제어합니다.

  • 워크그룹 조작: 프로그램 선택, 시작, 정지, 리셋 등의 명령을 시운전 및 테스트에 활용
  • 축 분리: 문제가 발생한 특정 축만 일시적으로 작업에서 제외하고 생산을 계속할 수 있도록 하여 가동률(Uptime)을 극대화하는 핵심 기능
💡 핵심 가치: ENRZ-DP50은 단순한 디스플레이가 아닌, 생산 효율성과 품질 관리를 한 차원 높이는 현장 운영의 통합 핵심 솔루션입니다.

⚙️ 2. 시스템 구성 및 하드웨어

ENRZ-DP50은 마스터 컨트롤러(MU50)와 작업자 간의 핵심 접점 역할을 합니다. 직관적인 터치스크린을 통해 시스템 상태를 파악하고 명령을 내립니다.

시스템 연결 방식

ENRZ-DP50은 시스템 환경에 따라 두 가지 방식으로 MU50과 연결됩니다.

🌐 Ethernet 연결

  • CAT.5E 이상 스트레이트 LAN 케이블 사용
  • HUB를 통한 다중 장비 연결 가능
  • 1:1 직접 연결 지원

🔌 RS-422 직렬 연결

  • 전용 통신 케이블 사용 (ENRZ-CVDP2-□□□)
  • 안정적인 1:1 통신 구성
  • 노이즈에 강한 직렬 통신

주요 부분 명칭 및 기능

명칭 핵심 기능
상태 LED 운전 상태를 색상으로 표시 (녹색: 운전 중, 적색: 전원 기동 시 등)
시리얼 인터페이스 (COM2) RS-422 직렬 통신 케이블 연결 포트
전원 플러그 커넥터 장치 구동을 위한 DC24V 전원 공급
이더넷 인터페이스 Ethernet 통신용 표준 RJ-45 포트 (10BASE-T/100BASE-TX)

🔩 3. 설치 및 배선 가이드

장비의 높은 성능과 안정적인 운영을 보장하기 위해 올바른 설치 절차를 준수해야 합니다.

패널 설치 절차

  1. 사전 조건 확인: 패널 두께 1.6mm ~ 5mm, 주변 공간 확보 상하좌우 100mm 이상, 설치 각도 수직 기준 30° 이내
  2. 패널 가공 및 방수 패킹 확인 (IP65f 등급 보장)
  3. DP50 삽입 및 후면 고정: 고정쇠 장착 및 체결, 체결 토크 0.5 N·m (대각선 방향 균등 체결)

전원 연결 및 접지

⚠️ 중요: 안정적인 전원 공급과 접지는 노이즈 방지 및 장비 보호에 필수입니다.

DC24V 전원 연결: 권장 전선 규격(0.75 ~ 2.5 mm²)에 맞게 전용 커넥터의 +, -, FG 단자에 정확히 결선합니다.

접지 방식 평가 설명
전용 접지 ✓ 최선 (권장) DP50의 FG 단자를 다른 장비와 공유하지 않고 독립적으로 접지
공용 접지 ○ 양호 제3종 접지(100Ω 이하) 수준의 안정적인 공용 접지점 이용
건너뛰기 접지 ✗ 금지 하나의 접지선으로 여러 장비를 직렬 연결하는 방식은 절대 금지
🚫 절대 금지: 건너뛰기 접지(데이지 체인) 방식은 과부하 및 보호 미흡으로 절대 사용하지 마십시오.

🖥️ 4. 초기 시스템 설정

장치의 모든 핵심 기능을 사용하기 위한 필수 단계입니다.

연결 방식별 초기 설정

연결 방식 모델 설정 절차 요약
Ethernet ENRZ-DP50-E
  1. MU50 전원 ON 및 시스템 파라미터(IP, 서브넷) 설정
  2. DP50 전원 ON → 오프라인 메뉴 → [본체 설정] > [이더넷 설정]에서 IP/마스크/게이트웨이 설정
  3. [주변기기 설정]에서 통신할 MU50의 IP 주소 입력 후 재부팅
RS-422 ENRZ-DP50-S
  1. 전용 통신 케이블로 DP50의 COM2와 MU50의 PANEL 포트 연결
  2. 양쪽 장치 전원 ON 시 자동으로 연결 완료

최초 기동 시 보안 설정 해제

공장 출하 시 잠금 상태인 주요 기능을 사용하기 위해 보안을 해제해야 합니다.

  1. 초기 화면 우측 하단 보안 버튼 터치
  2. 초기 비밀번호 1234 입력
  3. 보안 설정 화면에서 '파라미터 편집'과 '오퍼레이션' 항목을 '금지'에서 '허가'로 변경
